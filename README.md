# 各种 Configs
主要是我为了自己需要在其他机器上复用一些配置过的 tools 而记录的各种操作流程. 也可以视作半个教程 (基本都有详细的说明.)
- **/Shells**: bash, zsh, fish 的配置文件
- **/Latex**: 好用的模板 (我 fine tune 过的 elegantbook).
- **/Editors** (和 IDEs):我的 VSCode 和 vim, nvim 的配置文件和使用方法
- **/OSconfigs**: 在新机器上完整地安装 win11 和 Ubuntu desktop 双系统的流程, 配置 Ubuntu 的指令, 各种我的自定义设置, 和 crash consistency 方法等等; 以及我的 macOS 的一些配置 (我倾向于把它 windowize. 以及利好于 Latex & Markdown writing 的一些配置.)
