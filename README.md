# 各种 Configs
主要是我为了自己需要在其他机器上复用一些配置过的 tools 而记录的各种操作流程. 也可以视作半个教程 (基本都有详细的说明.)
- **/Shells**: bash, zsh, fish 的配置文件
- **/Editors**:我的 VSCode, Cursor, vim, nvim 的配置文件和使用方法
- **/OSconfigs**: 在新机器上安装并配置一些 Linux 发行版 (目前只有 UBuntu) 的流程, 以及我的一些自定义设置; 我的 macOS 的配置 (我倾向于把它 windowize 以及利好于 Latex & Markdown writing).
